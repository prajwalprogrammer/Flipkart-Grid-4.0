# Code
Following commands will be of utmost importance while cloning the repository
1. `npm i`
2. `npx hardhat node`
3. `npx hardhat run src/backend/scripts/deploy.js --network localhost`
4. `npm run start`

Change Directory to server folder <br/>
1. `cd src/server/` <br/>
2. `npm i ` <br/>
3. `node index.js` <br/>
